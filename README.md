# Momus Buffs

Decreases Momus's cooldown by 0.5 seconds for ranks II through V. Rank I/"Common Rarity" is not affected.


## Configuration

- enabled - Enable or disable the mod.
  - `Default: true`

- HephLegendaryBonus - Whether Hephaestus' Legendary boon is buffed (0.5 seconds instead of 1.0 seconds).
  - `Default: false`
